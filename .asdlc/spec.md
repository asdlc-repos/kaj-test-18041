# Overview

An internal leave management application where WSO2 employees can request time off and their managers can review and respond to those requests.

# Personas

- Employee — submits leave requests and checks their balance
- Manager — reviews and decides on team members' requests

# Features

- Employees request leave by selecting dates and a leave type
- Managers see requests from their direct reports
- A manager approves or rejects a request with an optional note
- Employees view the status of their submitted requests
- Everyone sees their available leave balance by type
- Employees can cancel a pending request before it's decided
